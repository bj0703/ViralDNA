## Context

中栏现在只是简单占位内容，需要升级为剪映风格专业视频创作主工作区。顶部三标签页导航，三个标签页各自完全独立的专属内容：样例视频页带时间轴+详情信息、素材视频页展示素材列表+槽位匹配状态、结果视频页显示单条拼接轨道。完全复用已有的 Soft Lavender 紫蓝主题，和左右栏风格保持统一，不用任何第三方视频编辑依赖包。

## Goals / Non-Goals

**Goals:**
- 中栏最顶部三标签页导航：样例视频 / 素材视频 / 结果视频，点击切换不同标签页。
- 样例视频标签页：视频播放器 + 时间轴线 + 节点详情信息展示。
- 素材视频标签页：素材列表 + 每条素材信息 + 槽位匹配状态展示。
- 结果视频标签页：单条拼接视频轨道，展示最终拼接完成的视频信息。

**Non-Goals:**
- 不引入任何第三方视频编辑/时间轴 npm 包，完全自己手写纯 React + CSS 实现。
- 不在第一版实现真实视频文件解码播放，先用模拟占位 + 封面图代替。

## Decisions

### 1. 三标签页完全独立，三个不同专属内容

```
┌─────────────────────────────────────────────────────────────────────┐
│ [样例视频]  [素材视频]  [结果视频]     ← 三标签页导航              │
├─────────────────────────────────────────────────────────────────────┤
│  👇  点击不同标签显示完全不同的内容 👇                               │
└─────────────────────────────────────────────────────────────────────┘
```

### 2. 标签1：样例视频专属内容

```
┌─────────────────────────────────────────────┐
│                                           │
│          样例视频播放器大封面                 │
│  ▶  00:00 / 01:30  播放控制条           │
│                                           │
│  🕒 00:00  ──── 时间轴节点 ─── 01:30    │
│  ● ● 🟦 ●  可点击圆点节点                   │
│          [ 点击后详情卡片 ]                │
└─────────────────────────────────────────────┘
```

内容：视频播放器 + 带可点击圆点节点的时间轴线，点击节点弹出该点的具体详情信息卡片。

### 3. 标签2：素材视频专属内容

```
┌─────────────────────────────────────────────────────────────┐
│  素材列表                                                    │
│  ┌─────────────────────────────────────────────────────┐  │
│  │ 🎬 素材A.mp4  [ 槽位匹配: ✅ 已分配 ]             │  │
│  │ 🎬 素材B.mp4  [ 槽位匹配: ⏳ 匹配中 ]             │  │
│  │ 🎙️ 音频A.mp3  [ 槽位匹配: ✅ 已分配 ]             │  │
│  └─────────────────────────────────────────────────────┘  │
│  槽位匹配状态汇总：3/5 已完成                               │
└─────────────────────────────────────────────────────────────┘
```

内容：纵向素材列表，每条素材显示文件名 + 槽位匹配状态徽章（已分配✅/匹配中⏳），底部显示整体进度汇总。

### 4. 标签3：结果视频专属内容

```
┌─────────────────────────────────────────────────────────────┐
│  最终拼接视频                                                │
│                                                           │
│  🎬  [====片段A====][=片段B=][==片段C==]  ← 单条轨道      │
│                                                           │
│  导出总时长: 02:15  分辨率: 1080p  输出进度: 100%        │
│                                                           │
│  [ 下载导出文件 ]  [ 重新编辑 ]                             │
└─────────────────────────────────────────────────────────────┘
```

内容：单条拼接视频轨道，横向铺满展示所有拼接完成的片段块，下方显示导出信息和操作按钮。

决定理由：三个标签页内容完全独立互不干扰，用户在不同工作流阶段看到完全对应的专属内容，体验非常清晰。

## Risks / Trade-offs

- [三个标签页状态内存占用] → Mitigation：切换非激活标签时卸载不必要的大组件，保持性能轻量。

## Migration Plan

1. 实现三标签页导航切换组件。
2. 实现样例视频标签页内容（播放器 + 时间轴节点）。
3. 实现素材视频标签页内容（素材列表 + 槽位匹配状态）。
4. 实现结果视频标签页内容（单条拼接轨道）。
5. 整体拼装替换原有中栏占位组件。

## Open Questions

- 三个标签页完全独立，互不共享子组件，结构清晰。
