## ADDED Requirements

### Requirement: 工作台明确展示是否为真实视频分析
系统 SHALL 在分析工作台中明确展示当前任务是否真正基于视频输入完成分析。

#### Scenario: 加载真实视频分析结果
- **WHEN** 工作台加载一个真实视频分析任务结果
- **THEN** 用户能看到当前 provider、视频输入形态和分析指令

#### Scenario: 加载非视频降级结果
- **WHEN** 工作台加载一个回退到非视频输入路径的结果
- **THEN** 用户能明确看到这不是等价的真实视频分析结果

### Requirement: 工作台展示视频分析失败原因
系统 SHALL 在工作台状态区或结果区展示真实视频分析失败、能力不支持或显式降级的原因。

#### Scenario: endpoint 不支持视频输入
- **WHEN** 当前任务未能进入真实视频分析 API
- **THEN** 工作台展示失败原因，并帮助用户判断是配置问题还是能力边界
