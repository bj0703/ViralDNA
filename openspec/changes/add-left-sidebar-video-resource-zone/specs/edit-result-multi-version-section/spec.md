## ADDED Requirements

### Requirement: 系统提供剪辑结果多版本保存板块
系统 SHALL 在左栏最底部展示剪辑结果多版本独立板块，纵向时序列表展示历史版本。

#### Scenario: 板块空状态展示
- **WHEN** 用户尚未生成任何剪辑结果版本
- **THEN** 板块显示空状态提示文字「暂无剪辑版本」

#### Scenario: 新增剪辑版本记录
- **WHEN** 系统新生成一个剪辑导出结果
- **THEN** 新的版本项出现在列表最顶部，显示版本号（Vx.x）、生成时间戳、预览按钮和删除按钮

#### Scenario: 删除指定历史版本
- **WHEN** 用户点击某个剪辑版本行的删除按钮
- **THEN** 该历史版本记录被移除，其余版本保持有序排列
