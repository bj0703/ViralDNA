## ADDED Requirements

### Requirement: 系统支持格式化结果卡片渲染
系统 SHALL 渲染 Agent 任务完成后的格式化结果卡片， SHALL 根据不同 cardType 渲染对应样式的结构化布局。

#### Scenario: Agent 完成任务输出结果卡片
- **WHEN** 某个 Agent 的执行任务全部完成
- **THEN** 该 Agent 消息的最终结果区渲染对应的格式化结果卡片，卡片圆角为 16px，带极浅阴影

### Requirement: 系统支持多卡片横向翻页浏览
系统 SHALL 当有多个结果卡片生成时， SHALL 提供横向翻页控件让用户在不同卡片之间切换浏览。

#### Scenario: 翻页控件展示当前页数
- **WHEN** 存在大于 1 个结果卡片
- **THEN** 卡片顶部显示翻页指示器，展示「结果卡片 (N / 总数)」字样，左右两侧有向左/向右翻页箭头按钮

#### Scenario: 点击右箭头切换下一张卡片
- **WHEN** 用户点击向右翻页箭头按钮，且当前不是最后一张卡片
- **THEN** 横向滚动切换到下一张结果卡片展示

#### Scenario: 点击左箭头切换上一张卡片
- **WHEN** 用户点击向左翻页箭头按钮，且当前不是第一张卡片
- **THEN** 横向滚动切换回上一张结果卡片展示
